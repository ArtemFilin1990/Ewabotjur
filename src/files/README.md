# Files

## Назначение

Модуль обработки файлов различных форматов (PDF, DOCX, TXT, MD, изображения).

## Основные компоненты

- Загрузка файлов из Telegram
- Парсинг PDF и DOCX
- OCR для изображений
- Извлечение текста из документов
- Генерация выходных файлов (DOCX, XLSX, MD)

## Поддерживаемые форматы

**Входные:**
- PDF
- DOCX
- TXT
- MD
- Изображения (с OCR)

**Выходные:**
- DOCX
- XLSX
- MD
