# Canonical Prompts

## Назначение

Хранилище канонических промптов для 9 специализированных юридических сценариев.

## Структура

Каждый сценарий представлен отдельным файлом с:
- Системным промптом
- Шаблоном ответа
- Примерами форматирования
- Критериями качества выходных данных

## Сценарии

1. `legal_document_structuring.md` - Структурирование юридического документа
2. `dispute_preparation.md` - Подготовка к спору
3. `legal_opinion.md` - Юридическое заключение
4. `client_explanation.md` - Объяснение для клиента
5. `claim_response.md` - Ответ на претензию
6. `business_context.md` - Контекст деловой переписки
7. `contract_agent_rf.md` - Агент по договорам (РФ)
8. `risk_table.md` - Таблица рисков
9. `case_law_analytics.md` - Анализ судебной практики
