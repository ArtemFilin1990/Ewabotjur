# Database

## Назначение

Модуль работы с базой данных для хранения сессий, истории и метаданных.

## Основные компоненты

- Подключение к MongoDB
- Модели данных (сессии, пользователи, история запросов)
- Миграции
- Индексы для оптимизации запросов

## Коллекции

### Sessions
- Хранение состояния сессий пользователей
- TTL для автоматической очистки
- История сообщений в рамках сессии

### Users
- Профили пользователей
- Telegram ID
- Настройки и предпочтения

### History
- Архив выполненных запросов
- Метаданные обработки
- Ссылки на артефакты

## Конфигурация

- Connection string
- Pool settings
- Индексы и TTL
