# LLM

## Назначение

Модуль взаимодействия с языковыми моделями (LLM) для обработки юридических запросов.

## Основные компоненты

- Клиенты для различных LLM API (OpenAI, Anthropic, YandexGPT и др.)
- Управление промптами
- Обработка ответов
- Retry-логика и обработка ошибок
- Мониторинг токенов и стоимости

## Конфигурация

- Выбор модели
- Настройка параметров (temperature, max_tokens)
- Таймауты и повторные попытки
