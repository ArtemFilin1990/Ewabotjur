# Bot

## Назначение

Модуль для работы с Telegram Bot API. Содержит обработчики команд, сообщений и колбэков от пользователей.

## Основные компоненты

- Инициализация бота
- Обработчики команд (/start, /help и др.)
- Обработчики текстовых и файловых сообщений
- Middleware для логирования и контроля доступа

## Зависимости

- aiogram или python-telegram-bot
- Модуль router для определения сценариев
