# Domain

## Назначение

Модуль бизнес-логики и доменных моделей проекта.

## Основные компоненты

### Модели (DTO)
- User - модель пользователя
- Session - сессия взаимодействия
- Scenario - тип юридического сценария
- Document - входной/выходной документ
- Request - запрос пользователя
- Response - ответ системы

### Бизнес-логика
- Валидация входных данных
- Оркестрация сценариев
- Применение бизнес-правил
- Расчет метрик и оценок

### Константы
- Типы сценариев
- Форматы документов
- Пороговые значения уверенности
- Лимиты и ограничения

## Принципы

- Независимость от инфраструктуры
- Чистые функции для бизнес-логики
- Явные типы и валидация
