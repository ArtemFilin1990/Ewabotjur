# Router

## Назначение

Модуль маршрутизации запросов для определения подходящего юридического сценария.

## Основные компоненты

- Детерминированное определение сценария с порогом уверенности 75%+
- Классификация входящих запросов
- Маппинг на один из 9 канонических сценариев

## Сценарии

1. Структурирование юридического документа
2. Подготовка к спору
3. Юридическое заключение
4. Объяснение для клиента
5. Ответ на претензию
6. Контекст деловой переписки
7. Агент по договорам (юрисдикция РФ)
8. Таблица рисков
9. Анализ судебной практики
